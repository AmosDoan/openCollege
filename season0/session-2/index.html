### Flask로 Web Application Server * Bot 개발을 위한 첫걸음
### URL의 형식 ``` http://localhost/서울시/송파구?동=잠실동 ``` * localhost 이하를 Path라 한다. * 송파구 뒤의 물음표 이하를 Query Parameter라 한다.
### Web API 만들기 ``` from flask import Flask from flask import request app = Flask(__name__) @app.route('/hello/<name>') def index(name): return "Hello " + name @app.route('/profile') def profile(): age = request.args.get('age') return "My age is " + age if __name__ == '__main__': app.run() ```
### 도전 과제 : Welcome! Web API만들기 * http://localhost:8000/{name} 호출시 다음과 같이 출력하는 홈페이지 ``` 환영합니다! ${name}의 홈페이지입니다! ``` * http://localhost:8000/{name}?family={count} 호출시 다음과 같이 출력하는 홈페이지 ``` ${name}님의 가족은 ${count}명입니다. ```
